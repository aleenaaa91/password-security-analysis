Password Security & Authentication Analysis 🔐

 📌 Project Overview
This project focuses on identifying security vulnerabilities in web applications, specifically related to password security and authentication mechanisms. The assessment was conducted using OWASP Top 10 methodology.

 🔍 Vulnerabilities Identified
- SQL Injection
- Broken Authentication
- Cross-Site Scripting (XSS)
- Cross-Site Request Forgery (CSRF)
- Security Misconfiguration

 🛠 Tools Used
- Nmap
- Burp Suite
- Web Browser (Manual Testing)

 🔧 Tool Developed
### Password Strength Analyzer (Python)

A simple tool developed to evaluate password strength based on:
- Minimum length
- Uppercase characters
- Numbers
- Special characters

 ▶️ How to Run the Tool
```bash
python3 password_tool.py
